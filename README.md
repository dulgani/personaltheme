This is Hexo theme

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/dulgani/personaltheme.git
$ cd personaltheme
$ npm install
```

Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server
```
